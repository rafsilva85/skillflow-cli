# SkillFlow CLI

[![Available on SkillFlow](https://raw.githubusercontent.com/rafsilva85/awesome-ai-skills/main/badges/skillflow-available.svg)](https://skillflow.builders)
[![PyPI version](https://img.shields.io/pypi/v/skillflow.svg)](https://pypi.org/project/skillflow/)

Search, discover, and install AI agent skills from the [SkillFlow](https://skillflow.builders) marketplace — directly from your terminal.

## Install

```bash
pip install skillflow
```

## Usage

```bash
# Search for skills
skillflow search docker
skillflow search "credit optimization"

# Show trending skills
skillflow trending

# Get skill details
skillflow info credit-optimizer-v5

# Install a skill
skillflow install credit-optimizer-v5

# List categories
skillflow categories

# List publishers
skillflow publishers
```

## What are AI Skills?

Skills are instruction files (`SKILL.md`) that give AI agents domain-specific knowledge and capabilities. They work across Claude Code, GitHub Copilot, Cursor, Gemini CLI, Windsurf, and more.
